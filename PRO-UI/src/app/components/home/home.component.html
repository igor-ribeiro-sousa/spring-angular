Sistema de Chamados Estudo Spring Boot + Angular

Olá! Seja bem vindo ao sistema de chamados desenvolvido por mim.
Esse é um sistema para gerenciamento de chamados desenvolvido em Angular 12 e Spring Boot 2. O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT e muito mais. O código fonte do projeto pode ser acessado clicando no link Github no menu lateral.
